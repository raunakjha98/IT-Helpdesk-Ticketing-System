# IT Helpdesk Ticketing System (Mini Project)

## Project Description
This is a simple IT Helpdesk Ticketing System built using Python and CSV file storage. The project simulates how IT support teams manage user issues, create tickets, and track ticket status.

## Features
- Create new IT support tickets
- View all existing tickets
- Update ticket status (Open, In Progress, Resolved, Closed)
- Store ticket data in CSV file

## Tools & Technologies
- Python
- CSV File Handling
- Basic File Management

## How to Run
1. Install Python
2. Download project files
3. Run:
   python ticket_system.py

## Use Case
This project simulates a real-world IT Helpdesk environment and is useful for learning IT support workflows and basic automation.

## What I Learned
- IT ticket lifecycle management
- File handling in Python
- Basic automation for IT support tasks
- Understanding of helpdesk operations
